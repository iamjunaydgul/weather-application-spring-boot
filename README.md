# weather-application-spring-boot
<b>Simple weather application build with spring boot and used apache camel weather component for getting weather information.</b><br>

This application is developed using Spring Boot, Apache Camel & its Weather Component. Used OpenWeatherMap Api to get weather information.<br>

<b>This project is based on the Spring Boot project and uses these packages:</b><br>
 1: Spring boot<br>
 2: Apache Camel<br>
 3: Apache Camel Weather Component<br>
 4: Thymeleaf<br>
 5: mysql<br>
 6: h2 database<br>
 7: Jquery datatable<br>
 8: OpenWeatherMap Weather API<br>
